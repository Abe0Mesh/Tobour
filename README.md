# Tobour

This is a top down RPG game made by Abraham Meshesha

## Status
Early Development,
Currently only texted based and player introduction

## Roadmap
Next step is to implement proper game states, then actions handling.


## Setup
1. Run 'mvn clean package'
2. Run 'mvn exec:java'

## Project Structure
- 'src/main/java' main game code
- 'src/tests/java' tests
